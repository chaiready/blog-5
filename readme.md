
## Free 社区/论坛/博客
- 2019-7-7
- 卢沟桥事变纪念日，每一位浴血奋战的英雄将士永垂不朽
- laravel + layui
- laui社区免费模板（感谢）
- 响应式页面，适应多个终端
- 内置websocket,实现即时通讯私聊、群聊
- 暂不支持离线消息
- 启动websocket，windows环境双击start_for_win.bat,linux环境 命令 php start.php start
- 安装注意：将.env 数据库地址及Email改为你自己的数据
- 遇到安装问题可联系 468027411@qq.com
- 没写管理后台
- 遇到BUG或漏洞，欢迎赐教指正
- 如果可以点个Star,持续更新
